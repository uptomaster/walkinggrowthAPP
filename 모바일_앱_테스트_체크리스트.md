# 모바일 앱 테스트 체크리스트

## ✅ 빌드 완료

APK 위치:
```
android/app/build/outputs/apk/debug/WalkStory-debug.apk
```

## 📱 APK 설치 방법

### 방법 1: USB 디버깅 (권장)

1. 안드로이드 기기에서 **개발자 옵션** 활성화
2. **USB 디버깅** 활성화
3. USB로 PC에 연결
4. 다음 명령어 실행:
```bash
adb install android/app/build/outputs/apk/debug/WalkStory-debug.apk
```

### 방법 2: 직접 전송

1. APK 파일을 안드로이드 기기로 전송 (이메일, 클라우드 등)
2. 기기에서 파일 관리자로 APK 열기
3. **알 수 없는 출처** 허용 (필요시)
4. 설치 진행

## 🔍 디버깅 방법

### 안드로이드 스튜디오 Logcat 사용

1. 안드로이드 스튜디오 열기
2. 하단 **Logcat** 탭 클릭
3. 필터: `WalkStory` 또는 `MainActivity`
4. 로그 확인:
   - `OAuth callback URL`
   - `handleOAuthCallback`
   - `API_BASE`
   - 네트워크 오류 메시지

### 주요 로그 확인 사항

```
MainActivity: Received intent with scheme: walkstory, host: oauth
MainActivity: OAuth callback URL: walkstory://oauth?token=...
handleOAuthCallback called with URL: walkstory://oauth?token=...
API_BASE 설정 완료: https://walkinggrowth-app.vercel.app
```

## ⚠️ 문제 해결

### 카카오 로그인이 작동하지 않으면

1. **카카오 개발자 콘솔 확인**
   - 네이티브 앱키: Android 플랫폼 등록 확인
   - 리다이렉트 URI 등록 확인

2. **앱 재설치**
   ```bash
   adb uninstall com.walkstory.app
   adb install android/app/build/outputs/apk/debug/WalkStory-debug.apk
   ```

3. **로그 확인**
   - Logcat에서 `MainActivity` 로그 확인
   - `handleOAuthCallback` 호출 여부 확인

### 일반 로그인이 작동하지 않으면

1. **네트워크 확인**
   - 인터넷 연결 확인
   - Vercel 서버 상태 확인

2. **API_BASE 확인**
   - Logcat에서 `API_BASE 설정 완료` 로그 확인
   - 값이 `https://walkinggrowth-app.vercel.app`인지 확인

3. **서버 로그 확인**
   - Vercel 대시보드에서 함수 로그 확인
