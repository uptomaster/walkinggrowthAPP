## 🔍 네이티브 앱키 페이지 찾기

### 방법 1: 현재 페이지에서 찾기

현재 보고 계신 "플랫폼 키" 페이지에서:

1. **상단 탭 확인**
   - "JavaScript 키" 탭 옆에 **"네이티브 앱키"** 탭이 있는지 확인
   - 있으면 클릭!

2. **페이지 내 섹션 확인**
   - "JavaScript 키" 섹션 아래에 **"네이티브 앱키"** 섹션이 있는지 확인
   - 또는 왼쪽 사이드바에서 "네이티브 앱키" 메뉴 확인

### 방법 2: 직접 URL 접근

브라우저 주소창에 다음 URL을 입력:

```
https://developers.kakao.com/console/app/1f0b84df4d86cb3d668020799bcc7473/platform-key/native
```

또는:

```
https://developers.kakao.com/console/app/1f0b84df4d86cb3d668020799bcc7473/platform-key?tab=native

## 📝 네이티브 앱키 등록 방법

"네이티브 앱키" 페이지를 찾으면:

2. 다음 정보 입력:

**패키지 이름:**
```

**키 해시:**
```
61F95D5037CA3D777992B6EAD6625D5C9F2E048
```
⚠️ **중요**: 콜론(:)을 제거한 형식으로 입력해야 합니다!

3. **"저장"** 또는 **"등록"** 버튼 클릭

## 🎯 네이티브 앱키 페이지 특징

네이티브 앱키 페이지에서는 보통:
- **Android**, **iOS** 플랫폼 목록이 보입니다
- 각 플랫폼 옆에 **"추가"**, **"등록"**, 또는 **"+"** 버튼이 있습니다
- Android를 선택하면:
  - **패키지 이름** 입력 필드
  - **키 해시** 입력 필드 (여러 개 추가 가능)
  - **저장** 버튼

---

## ⚠️ 중요 사항

- **패키지 이름**: 정확히 `com.walkstory.app` 입력 (대소문자 구분)
- **키 해시**: 콜론(:)을 제거한 형식으로 입력 (예: `61F95D5037CA3D777992B6EAD6625D5C9F2E048`)
- **현재 키 해시**: 디버그 키스토어용 (APK 테스트용)
- **나중에**: 프로덕션 빌드용 키 해시도 추가로 등록 필요
- **여러 키 해시**: 여러 개발자의 디버그 키 해시를 모두 등록할 수 있습니다

- "플랫폼 키" 페이지는 JavaScript 키와 네이티브 앱키를 모두 관리하는 페이지입니다
- JavaScript 키는 웹용, 네이티브 앱키는 Android/iOS 앱용입니다
- Android 앱에서 카카오 로그인을 사용하려면 네이티브 앱키에 Android 플랫폼을 등록해야 합니다
