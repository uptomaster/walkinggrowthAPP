# 카카오 개발자 콘솔에서 플랫폼 등록 메뉴 찾기

## 🔍 플랫폼 등록 메뉴 찾는 방법

카카오 개발자 콘솔 UI가 업데이트되면서 메뉴 위치가 변경되었을 수 있습니다.

### 방법 1: 앱 설정에서 찾기

1. 카카오 개발자 콘솔 접속
2. **내 애플리케이션** → **WalkStory** 앱 선택
3. 왼쪽 사이드바에서 **"앱 설정"** 클릭
4. 아래 메뉴들을 확인:
   - **플랫폼** ← 여기일 가능성 높음
   - **플랫폼 설정**
   - **앱 플랫폼**
   - **플랫폼 등록**

### 방법 2: 제품 설정에서 찾기

1. 왼쪽 사이드바에서 **"제품 설정"** 클릭
2. **"카카오 로그인"** 클릭
3. 그 안에 **"플랫폼"** 또는 **"플랫폼 설정"** 메뉴 확인

### 방법 3: 직접 URL 접속

카카오 개발자 콘솔 URL 구조:
```
https://developers.kakao.com/console/app/{앱키}/platform
```

또는:
```
https://developers.kakao.com/console/app/{앱키}/settings/platform
```

`{앱키}` 부분을 WalkStory 앱의 REST API 키로 변경해보세요.

### 방법 4: 검색 기능 사용

1. 카카오 개발자 콘솔 상단 검색창 클릭
2. **"플랫폼"** 또는 **"Android"** 검색
3. 관련 메뉴 찾기

---

## 📸 플랫폼 등록 페이지 특징

플랫폼 등록 페이지에서는 보통:
- **Android**, **iOS**, **Web** 탭 또는 목록이 보입니다
- 각 플랫폼 옆에 **"추가"** 또는 **"등록"** 버튼이 있습니다
- Android를 클릭하면:
  - **패키지 이름** 입력 필드
  - **키 해시** 입력 필드
  - **저장** 버튼

---

## ⚠️ 플랫폼 키 페이지와의 차이

### 플랫폼 키 페이지 (현재 보고 계신 페이지)
- REST API 키, JavaScript 키, 네이티브 앱 키 관리
- **여기가 아닙니다!**

### 플랫폼 등록 페이지 (찾아야 할 페이지)
- Android, iOS, Web 플랫폼 등록
- 패키지 이름과 키 해시 입력
- **여기입니다!**

---

## 💡 여전히 찾기 어려우시면

카카오 고객센터나 문서를 참고하세요:
- [카카오 개발자 문서](https://developers.kakao.com/docs)
- 카카오 개발자 콘솔 하단의 **"문서"** 또는 **"도움말"** 링크

또는:
1. 카카오 개발자 콘솔에서 **"문서 확인"** 버튼 클릭
2. "플랫폼 등록" 또는 "Android 플랫폼" 검색

---

## 🎯 빠른 확인 방법

카카오 개발자 콘솔에서:
1. 왼쪽 메뉴를 하나씩 클릭해보기
2. "Android" 또는 "플랫폼" 키워드가 있는 메뉴 찾기
3. 그 메뉴에서 패키지 이름과 키 해시를 입력할 수 있는 페이지 찾기
