## 🔗 직접 URL 접근 방법

### 방법 1: 네이티브 앱키 페이지 직접 접근 (가장 가능성 높음!)

카카오 개발자 문서에 따르면, Android 플랫폼 등록은 **"플랫폼 키" → "네이티브 앱키"**에서 합니다!

브라우저 주소창에 다음 URL을 입력하세요:

```
https://developers.kakao.com/console/app/1f0b84df4d86cb3d668020799bcc7473/platform-key/native
```

또는:

```
https://developers.kakao.com/console/app/1f0b84df4d86cb3d668020799bcc7473/platform-key?tab=native
```

### 방법 2: 플랫폼 등록 페이지 직접 접근

```
https://developers.kakao.com/console/app/1f0b84df4d86cb3d668020799bcc7473/platform
```

또는:

```
https://developers.kakao.com/console/app/1f0b84df4d86cb3d668020799bcc7473/settings/platform
```

### 방법 2: 앱 설정에서 찾기

1. 카카오 개발자 콘솔 메인 페이지로 이동
2. **내 애플리케이션** → **WalkStory** 앱 클릭
3. 왼쪽 사이드바에서 **"앱"** 섹션 확인
4. **"앱"** 섹션을 확장하면:
   - 일반
   - 플랫폼 키 ← (현재 여기)
   - 어드민 키
   - **플랫폼** ← (여기를 찾아야 함!)
   - 기타 메뉴들

### 방법 3: 카카오 로그인 설정에서 찾기

1. 왼쪽 사이드바에서 **"제품 설정"** 클릭
2. **"카카오 로그인"** 클릭
3. 그 안에 **"플랫폼"** 또는 **"플랫폼 설정"** 메뉴 확인

```
com.walkstory.app
```

**키 해시:**
```
61:1F:95:D5:03:7C:A3:D7:77:99:2B:6E:AD:66:25:D5:C9:F2:E0:48
```

3. **"저장"** 또는 **"등록"** 버튼 클릭

## 🔍 플랫폼 등록 페이지 특징

플랫폼 등록 페이지에서는 보통:
- **Android**, **iOS**, **Web** 탭 또는 목록이 보입니다
- 각 플랫폼 옆에 **"추가"**, **"등록"**, 또는 **"+"** 버튼이 있습니다
- Android를 선택하면:
  - **패키지 이름** 입력 필드
  - **키 해시** 입력 필드
  - **저장** 버튼
