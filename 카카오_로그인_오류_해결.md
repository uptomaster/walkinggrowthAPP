# 카카오 로그인 시작 실패 오류 해결

## 🔴 문제 상황

모바일 앱에서 카카오 로그인 시 "카카오 로그인 시작에 실패했어요" 오류 발생

## ✅ 해결 완료

### 1. Vercel 서버리스 함수 경로 구조 수정

**변경 전:**
- `api/auth/kakao-oauth-start.js` → `/api/auth/kakao-oauth-start` (잘못된 경로)

**변경 후:**
- `api/auth/kakao-oauth/start.js` → `/api/auth/kakao-oauth/start` ✅
- `api/auth/kakao-oauth/callback.js` → `/api/auth/kakao-oauth/callback` ✅

### 2. 리다이렉트 URI 수정

**변경 전:**
- `/api/auth/kakao-callback`

**변경 후:**
- `/api/auth/kakao-oauth/callback`

## 📝 카카오 개발자 콘솔 설정 업데이트 필요

리다이렉트 URI를 업데이트해야 합니다:

1. [카카오 개발자 콘솔](https://developers.kakao.com/) 접속
2. **내 애플리케이션** → **WalkStory** 앱 선택
3. **제품 설정** → **카카오 로그인** 클릭
4. **Redirect URI 등록** 섹션에서:
   - 기존 URI 삭제: `https://walkinggrowth-app.vercel.app/api/auth/kakao-callback`
   - 새 URI 추가: `https://walkinggrowth-app.vercel.app/api/auth/kakao-oauth/callback`

## 🚀 다음 단계

1. **Vercel에 배포**
   - 변경사항을 Git에 커밋하고 푸시
   - Vercel이 자동으로 배포

2. **카카오 개발자 콘솔 업데이트**
   - 리다이렉트 URI 변경 (위 참고)

3. **모바일 앱 재테스트**
   - APK 재설치 (선택사항)
   - 카카오 로그인 다시 시도

## 🔍 확인 사항

### Vercel 배포 확인

배포 후 다음 URL이 작동하는지 확인:
```
https://walkinggrowth-app.vercel.app/api/auth/kakao-oauth/start
```

예상 응답:
```json
{
  "authUrl": "https://kauth.kakao.com/oauth/authorize?...",
  "state": "..."
}
```

### 로그 확인

Vercel 대시보드에서 함수 로그 확인:
- `/api/auth/kakao-oauth/start` 함수 호출 여부
- 오류 메시지 확인

## ⚠️ 중요 사항

- **리다이렉트 URI 변경 필수**: 카카오 개발자 콘솔에서 반드시 업데이트해야 합니다
- **Vercel 배포 필요**: 변경사항이 Vercel에 배포되어야 합니다
- **테스트**: 배포 완료 후 모바일 앱에서 다시 테스트
