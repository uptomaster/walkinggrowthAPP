# Android 플랫폼 등록 직접 가이드

## ✅ 키 해시 (이미 생성 완료)
```
61:1F:95:D5:03:7C:A3:D7:77:99:2B:6E:AD:66:25:D5:C9:F2:E0:48
```

## 📍 카카오 개발자 콘솔에서 찾기

### 방법 1: 직접 URL 접근

카카오 개발자 콘솔에서 현재 보고 계신 URL을 확인하세요.

예를 들어:
```
https://developers.kakao.com/console/app/{앱키}/platform-key
```

이 URL에서 `platform-key` 부분을 `platform`으로 변경해보세요:
```
https://developers.kakao.com/console/app/{앱키}/platform
```

또는:
```
https://developers.kakao.com/console/app/{앱키}/settings/platform
```

`{앱키}` 부분은 WalkStory 앱의 REST API 키입니다.

### 방법 2: 사이드바 메뉴 확인

현재 이미지에서 보이는 사이드바에서:

1. **"제품 설정"** 클릭 (현재 collapsed 상태)
   - 클릭하면 확장됩니다
   - 그 안에 **"카카오 로그인"** 또는 **"플랫폼"** 메뉴가 있을 수 있습니다

2. **"앱"** → **"일반"** 클릭
   - 일반 설정 페이지에서 플랫폼 등록 옵션이 있을 수 있습니다

3. **"카카오 로그인"** (이미 expanded 상태)
   - 그 안에 **"플랫폼"** 또는 **"플랫폼 설정"** 메뉴가 있을 수 있습니다

### 방법 3: 검색 기능 사용

카카오 개발자 콘솔 상단의 검색창에서:
- **"Android"** 검색
- **"플랫폼 등록"** 검색
- **"패키지"** 검색

---

## 🎯 Android 플랫폼 등록 페이지 특징

플랫폼 등록 페이지에서는 다음과 같은 화면이 보입니다:

1. **플랫폼 목록 또는 탭**
   - Android
   - iOS  
   - Web

2. **Android 선택 시**
   - **패키지 이름** 입력 필드
   - **키 해시** 입력 필드
   - **저장** 버튼

---

## 📝 등록할 정보

플랫폼 등록 페이지를 찾으면:

1. **Android** 선택 또는 클릭
2. 다음 정보 입력:
   - **패키지 이름**: `com.walkstory.app`
   - **키 해시**: `61:1F:95:D5:03:7C:A3:D7:77:99:2B:6E:AD:66:25:D5:C9:F2:E0:48`
3. **저장** 클릭

---

## ⚠️ 중요 사항

- **패키지 이름**: 정확히 `com.walkstory.app` 입력 (대소문자 구분)
- **키 해시**: 콜론(:) 포함해서 입력
- **현재 키 해시**: 디버그 키스토어용 (APK 테스트용)
- **나중에**: 프로덕션 빌드용 키 해시도 추가로 등록 필요
